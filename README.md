## 常见问题与避坑指南

挖了这么久矿，我踩过的坑可能比你吃过的饭还多。这里分享几个新手最容易犯的错误。

**问题1：算力显示很低，或者经常掉线**

可能原因：网络不稳定、防火墙拦截、杀毒软件干扰。

解决办法：确保挖矿软件被加入防火墙和杀毒软件的白名单。如果用的是校园网或公司网络，可能会被限制，试试换个网络或用VPN。

**问题2：挖矿收益比预期低很多**

这个……得实话实说，挖矿收益本来就不高，别被网上那些"月入上万"的标题骗了。家用电脑挖矿，一天赚几块钱到几十块钱人民币是正常水平。

另外，收益会受币价波动影响。今天狗狗币涨了，你的收益就高；明天跌了，收益就低。这很正常。

**问题3：为什么我的显卡/CPU温度这么高？**

挖矿会让硬件满负荷运行，温度飙升是正常的。但如果GPU温度超过85°C，或者CPU超过90°C，就需要注意了。

**问题4：钱包地址填错了怎么办？**

如果还没开始挖矿，直接重新填一遍就行。如果已经挖了一段时间……那就比较麻烦了。unMineable不支持修改地址，币会打到错误的地址上，基本找不回来。所以填地址的时候一定要仔细检查。

unMineable是正规平台，运营好几年了，在Reddit、YouTube、各大加密货币论坛都有大量用户讨论。虽然也有人抱怨收益低（确实低），但不存在跑路或不付款的情况。
